# PEG修正因子模型（行业权重版）

## 一、核心公式

```text
目标PEG = 基础PEG × 环境因子 × 自身因子
```

其中：

```text
环境因子 = 环境加权平均分
自身因子 = 自身加权平均分
```

默认情况下，环境与自身内部均采用**等权平均**；但在不同行业中，允许根据商业本质进行**有限度的权重偏移**。

---

## 二、评分总原则

### 1. 统一打分方向
所有因子都按“**是否提升估值容忍度**”来打分：

- **1.15**：显著提升估值容忍度
- **1.00**：中性
- **0.85**：显著压低估值容忍度

对于名称带有负向含义的因子，采用反向理解：

- **政策风险越低，分数越高**
- **外部成本敏感性越弱，分数越高**
- **品类/客户集中度越健康，分数越高**

### 2. 分值范围
- 所有因子默认范围统一为：**0.85x–1.15x**
- 大多数情况尽量落在：**0.95x–1.05x**
- 只有有明确强证据时，才打到：**0.85x / 1.15x**

### 3. 证据优先
打分尽量基于以下证据，而不是主观印象：

- 财报与经营数据
- 行业竞争格局变化
- 终端需求和景气数据
- 政策、监管、关税、地缘事件
- 公司历史兑现能力与资本配置记录

---

## 三、分值档位锚点

为避免不同时间、不同公司之间打分漂移，统一使用以下锚点：

| 系数 | 定义 | 使用标准 |
|---|---:|---|
| **1.15** | 极强 | 显著优于同行，且有持续、明确、强证据支持；一般只用于极少数高确定性情形 |
| **1.10** | 明确偏强 | 明显优于行业平均，优势已被经营数据或行业事实验证 |
| **1.05** | 略强 | 比行业平均略好，有一定证据，但优势未到“显著”程度 |
| **1.00** | 中性 | 无明显加减分，或优劣并存后大致相抵 |
| **0.95** | 略弱 | 略低于行业平均，存在轻微拖累因素 |
| **0.90** | 明确偏弱 | 短板较明确，且已有较清晰证据验证 |
| **0.85** | 极弱 | 明显劣于同行，且问题已被反复验证或存在高确定性风险 |

### 使用细则

1. **默认从 1.00 出发**，再根据证据向上或向下微调。  
2. **没有强证据，不用 1.15 / 0.85**。  
3. **优先使用 1.05 / 0.95** 表达轻微倾向。  
4. 同一家公司应尽量保持前后口径一致，便于复盘。

## 四、因子体系

## 4.1 环境因子

| 因子 | 评定依据 | 需跟踪指标 |
|---|---|---|
| **竞争格局** | 行业集中度是否提升；头部企业份额是否扩大；是否存在明显价格战；新进入者是否强势；替代品是否增多；渠道资源是否被竞争对手争夺；新品能否迅速被同行复制；行业是否从成长期转向存量竞争 | 市占率、价格带变化、竞品投放、新品成功率 |
| **政策风险** | 是否受反垄断、监管趋严、数据合规、食品安全、环保、税收、关税、地缘政治影响；政策变化是否直接影响销量、价格、成本或扩张节奏；海外业务是否面临制裁、贸易限制、汇兑限制；是否存在牌照或资质不确定性 | 政策新闻、处罚记录、关税变化、监管态度 |
| **外部成本敏感性** | 利润是否高度受原材料、能源、运费、汇率、利率影响；成本上涨是否能顺利传导给终端价格；采购是否有锁价、长协机制；成本波动是否会造成毛利率剧烈变化；成本端是否受单一大宗品驱动 | 原材料占成本比、提价频率、毛利率弹性、汇率敞口 |
| **行业景气** | 行业是否处于扩容期；终端需求是加速、平稳还是走弱；行业渗透率是否提升；景气是否受宏观、地产、消费周期、补贴、库存周期影响；未来1–2年行业需求趋势是否清晰 | 行业增速、渗透率、库存周期、终端需求数据 |

## 4.2 自身因子

| 因子 | 评定依据 | 需跟踪指标 |
|---|---|---|
| **商业模式** | 是否具备品牌溢价；是否拥有渠道壁垒；是否具备提价能力；单位成本是否随规模扩大而下降；客户是否高频复购；产品是否容易被替代；毛利率是否长期稳定；费用投放减少后销量是否仍可维持 | 毛利率、渠道覆盖、复购率、单店产出 |
| **现金流质量** | 经营现金流是否持续匹配净利润；自由现金流是否长期为正；利润增长是否能转化为现金增长；营运资金是否持续恶化；应收账款和存货是否显著快于收入增长；现金流偏弱是扩产阶段性因素还是利润质量问题 | 经营现金流/净利润、FCF/净利润、应收周转、存货周转 |
| **再投资强度** | 维持增长是否依赖持续高CAPEX；是否需要持续高销售费用投放；新增产能、渠道、终端设备的回收期是否合理；边际投入产出比是否下降；扩张是否必须不断投入才能维持；增长是否具备自我强化能力 | CAPEX/收入、销售费用率、新增产能利用率、投入回收期 |
| **治理与资本配置** | 管理层是否兑现历史承诺；分红、回购、并购是否理性；是否频繁融资或摊薄股东；募集资金用途是否清晰；关联交易是否复杂；股权激励是否合理；资本开支是否有明确回报逻辑 | 分红率、回购、并购回报、再融资频率、股权激励摊薄 |
| **资产负债表** | 是否净现金；债务结构是否健康；短期偿债压力是否高；是否依赖外部融资维持扩张；财务费用是否可控；资产质量是否稳定；商誉、存货、应收是否存在减值风险；扩张后杠杆是否显著抬升 | 现金/债务、流动比率、利息保障倍数、商誉占比 |
| **品类/客户集中度** | 收入是否依赖单一大单品；利润是否高度集中在少数品类；是否依赖单一客户、单一区域、单一渠道；若核心品类放缓，是否有其他品类承接；集中度是下降还是上升 | 单品收入占比、前五客户占比、区域占比、渠道占比 |

## 五、默认权重设置（基础版）

### 5.1 环境因子默认等权

```text
环境因子 = (竞争格局 + 政策风险 + 外部成本敏感性 + 行业景气) ÷ 4
```

### 5.2 自身因子默认等权

```text
自身因子 = (商业模式 + 现金流质量 + 再投资强度 + 治理与资本配置 + 资产负债表 + 品类/客户集中度) ÷ 6
```

### 5.3 默认原则
- **优先使用等权版本**，保持模型简洁、稳定、可复盘。
- 只有在行业特征非常明确时，才使用“行业权重偏移版”。
- 权重偏移应当是**有限调整**，不要把模型变成高度主观的自由裁量工具。

---

## 六、行业权重偏移规则

## 6.1 使用原则

允许针对不同行业，对因子重要性进行**有限度权重偏移**。但应遵守以下规则：

1. **默认等权，偏移是例外，不是常态**。  
2. 单一因子的权重建议在默认权重基础上做**小幅调整**，通常不超过 **1.5倍**。  
3. 权重偏移必须符合行业商业本质，而不是为结论服务。  
4. 使用偏移版时，应在记录中写明：  
   - 为什么这个行业需要偏移  
   - 哪些因子被上调  
   - 哪些因子被下调  
5. 权重调整后，最终仍需做**标准化处理**，确保总权重为 100%。

## 七、加权计算方法

## 7.1 等权版

### 环境因子

```text
环境因子 = (竞争格局 + 政策风险 + 外部成本敏感性 + 行业景气) ÷ 4
```

### 自身因子

```text
自身因子 = (商业模式 + 现金流质量 + 再投资强度 + 治理与资本配置 + 资产负债表 + 品类/客户集中度) ÷ 6
```

---

## 7.2 行业权重偏移版

### 环境因子

```text
环境因子 = Σ(单项评分 × 权重)
```

### 自身因子

```text
自身因子 = Σ(单项评分 × 权重)
```

要求：

```text
环境权重合计 = 100%
自身权重合计 = 100%
```

### 最终公式

```text
目标PEG = 基础PEG × 环境因子 × 自身因子
```

## 八、总修正上限控制

虽然单项分值被限制在 **0.85–1.15**，但两组均值相乘后，仍可能放大总体偏差。为避免主观判断过度放大，建议加入总约束：

```text
总修正倍数 = 环境因子 × 自身因子
```

建议限制在：

```text
0.80 ≤ 总修正倍数 ≤ 1.20
```

更保守时，可使用：

```text
0.85 ≤ 总修正倍数 ≤ 1.15
```

### 含义
- 防止因为连续主观加分，导致最终PEG明显失真
- 防止因为短期悲观，把估值容忍度压得过低
- 让模型保持“修正”属性，而不是替代基础估值体系

## 九、实际使用流程

### Step 1：先定基础PEG
基础PEG应来自你对公司长期增速、行业位置、历史估值中枢的初步判断。

### Step 2：给10个因子逐项打分
先按统一锚点，从 **1.00** 出发微调。

### Step 3：决定是否采用行业权重偏移
- 若行业特征不强，直接使用**等权版**
- 若行业特征非常鲜明，使用**行业偏移版**

### Step 4：算出环境因子与自身因子
- 等权则直接平均
- 偏移版则按加权平均

### Step 5：算出目标PEG

```text
目标PEG = 基础PEG × 环境因子 × 自身因子
```

### Step 6：检查总修正是否过大
若总修正倍数超出预设上限，回头检查：
- 是否某些因子重复表达了同一件事
- 是否对短期变量打分过重
- 是否使用了过激档位

## 十一、一行版说明

先给 10 个因子按统一锚点打分；默认使用等权平均，只有在行业特征非常明确时才启用行业权重偏移；最后用：

并对总修正倍数做上限约束。
