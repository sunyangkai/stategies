# 中国宏观观察框架：近两年数据统计（2024-04 至 2026-03）

> 生成日期：2026-04-27  
> 数据范围：最近约 24 个月；季度指标覆盖 2024Q2—2026Q1。  
> 统计原则：月度发布的指标按月度列示；季度发布的指标按季度列示；部分官方只发布累计值的指标以累计同比列示。  
> 可信度处理：就业收入、宏观企业现金流质量已按你的要求剔除；库存、应收等只作为工业利润表的辅助项，不作为核心判断项。

---

## 0. 使用说明

本文件用于判断宏观是否进入“内生复苏”。核心不是看单个指标是否改善，而是看四条链条是否同时改善：

1. **价格链条**：PPI、CPI、核心CPI/服务CPI、外部原材料价格。
2. **需求链条**：社零、出口、进口、房地产销售。
3. **信用链条**：居民贷款、企业贷款、社融、M2、M1、M2-M1剪刀差。
4. **生产/投资/政策链条**：工业利润、制造业结构、民间投资、库存周期、产能利用率、财政支出/专项债。

判断口径：

> 如果 PPI、工业利润、M1 同时改善，但社零、居民贷款、民间投资、房地产、核心CPI/服务CPI没有同步改善，则更应解释为“价格端和资金端局部修复”，而不是“经济进入内生复苏”。

### 2.1 对价格和消费的判断

- **PPI**：2025年中低点约在 -3.6%，2026年3月转正至 +0.5%。价格端确实改善，但驱动可能包含输入型通胀、原材料价格上涨、部分供给约束和反内卷政策。
- **CPI**：整体温和，2026年2月受春节错位扰动明显，3月回落至1.0%。
- **社零**：2025年5月后持续走弱，2025年12月仅0.9%，2026年3月为1.7%，不支持“内需强复苏”判断。

### 3.3 信用层判断

- **总量流动性不紧**：M2、社融仍有支撑。
- **资金活化改善但不稳**：M1较2025年初改善，但2026年3月较2月回落，剪刀差略走阔。
- **居民部门仍弱**：2026Q1住户贷款显著少于2025Q1，短贷转负，中长期贷款少增。
- **企业端仍是信贷主力**：但企业贷款中短贷较强、中长期贷款并未明显加速，不能直接解释为民间投资信心恢复。

## 9. 汇率、外部价格、利率与信用利差

该组指标为解释变量，不直接代表内生复苏。建议在模型中作为“宏观扰动项”使用。

| 维度 | 建议跟踪指标 | 当前使用方式 |
|---|---|---|
| 汇率 | USD/CNY中间价、即期汇率 | 判断输入型通胀、出口利润和外资风险偏好 |
| 原油 | Brent/WTI月均价 | 判断PPI中石油链条的输入型压力 |
| 有色 | 铜、铝、锂、镍等价格 | 判断PPI中有色和新能源链条扰动 |
| 利率 | 10年国债收益率、DR007、LPR | 判断资金价格，而非直接判断需求强弱 |
| 信用利差 | AAA/AA产业债利差、城投利差 | 判断风险偏好和信用分层 |

> 注：这组高频数据不建议简单并入“内生复苏表”，因为其对PPI和资产价格影响很大，但不一定来自国内需求。

## 11. 投资与估值使用建议

1. **不要因为PPI转正就全面上调收入增速**：更适合先修正毛利率和价格假设，而非销量和需求假设。
2. **居民相关行业继续保守**：社零低位、居民贷款偏弱、地产链仍弱，不支持消费和地产链强复苏假设。
3. **制造业需结构化处理**：高技术、装备制造、部分反内卷行业可单独上修，但传统过剩产能行业不宜全面外推。
4. **宏观终局增速不宜上调**：当前更像“价格端局部修复 + 政策/外需支撑”，不是内生复苏确认。
5. **M1改善只能作为辅助信号**：只有在社零、居民贷款、民间投资、产能利用率同步改善时，才可提高对内生复苏的置信度。

## 12. 数据源与校验入口

优先级从高到低：

1. 国家统计局：PPI、CPI、社零、固定资产投资、房地产、工业利润、产能利用率

   - https://www.customs.gov.cn/
4. 财政部：财政收支、专项债、国债发行  
   - https://www.mof.gov.cn/
5. 东方财富Choice公开数据页：用于月度序列交叉校验
